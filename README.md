# 🚀 SRP Win Toolbox



**SRP Win Toolbox** is a Windows-native PowerShell GUI toolkit designed for **gamers, esports players, tournament competitors, and power users**.



It provides safe and reversible Windows optimization utilities that help improve system responsiveness by optimizing network settings, managing unnecessary background services, enhancing overall Windows performance, and safely removing pre-installed bloatware.



> **Supported Platform:** Windows 10 & Windows 11

> **Built With:** PowerShell + Windows Forms

> **License:** GPL-2.0



---



## ✨ Features



### 🌐 SRP Network & Smart DNS Tweaker

Optimize Windows networking for gaming and everyday use.

* Disable Nagle's Algorithm (TCP Delay)

* Optimize Network Throttling Index

* Automatic DNS Benchmark

* Smart DNS Selection

* Supports: Cloudflare, Google, Quad9, AdGuard DNS



### 🎮 SRP Tournament & Gaming Service Tweaker

Reduce unnecessary background services for better gaming performance.

* Disable Windows Telemetry & MapsBroker

* Optional SysMain, Windows Search, and Biometric Service Control

* One-click Restore to Default



### 🗑️ SRP Debloater

**New!** Safely remove unnecessary Windows pre-installed applications.

* Sleek Dark Mode UI (Cyberpunk style)

* **1-Click Bloatware Selection:** Pre-configured list for junk apps (Xbox, Bing, 3D Builder, etc.)

* Safe uninstallation with built-in error handling

* Lightweight and dependency-free



---



## 🛡 Safety

The toolkit only changes Windows configuration settings. It **does not** modify Windows system files or install third-party software. All changes are reversible via the built-in restore options.

## ⚡ Quick Run



Launch **PowerShell** and execute the following commands:



### SRP Debloater

```powershell

```



### Network Tweaker



```powershell

.\DNS_Tweaker.ps1

```



### Service Tweaker



```powershell

.\Service_Tweaker.ps1

```



> **Note:** Administrator permission is required to modify system settings.

## 🚀 Getting Started



1. Download the scripts from this repository.

2. Right-click the desired script (e.g., `SRP_Debloater.ps1`).

3. Select **Run with PowerShell**.

4. Accept the UAC prompt (Administrator permission).

5. Configure your settings and click **Apply** or **Uninstall**.
